Basic Integration

$$\int 7(x-2)^2 dx$$

$$\text{substitute } u = x-2, du = dx $$
$$= \int 7u^2du$$
used rule #1: $$=7 \int u^2du$$
used rule #4: $$=7 \frac{u^3}{3} + C$$
  • It can be solved by more than 1 solutions
  • Some integration needs more than 1 rule to solve
  • Changing it to the simpler form will make it be solved easier

Ways to Change it to Simpler Form

  • Expansion of Numerator (\((x+1)^2 = x^2 + 2x + 1\))
  • Separate Numerator (\(\frac{x+3}{x^2} = \frac{x}{x^2} + \frac{3}{x^2}\))
  • Complete a Square (\(\frac{1}{\sqrt{2x-x^2}} = \frac{1}{\sqrt{1-(x-1)^2}}\))
  • Addition or Subtraction (\(\frac{x}{x^2+2x+1} = \frac{x+1}{x^2+2x+1} - \frac{1}{x^2+2x+1}\))
  • Trigonometric Identity (\(sin^2x + cos^2x = 1\))

Integration By Parts

$$\int udv = uv - \int vdu + C$$

  • Select \(u\) and \(dv\) based on how hard the integral will become after choosing it (\(dv\) part must have \(dx\))
  • Guildline of choosing \(u\) (L-I-A-T-E) (With Exception):
    • L: Logarithmic function (\(lnx\))
    • I: Inverse Trigon function (\(arctanx\))
    • A: Algebriac function (\(x^n\))
    • T: Trigonometric function (\(sinx\))
    • E: Exponential function (\(e^x\))
  • if \(\int vdu\) part is still not trivial enough to solve with basic, you must do by part again with that part integration with the same \(u   \text{and}   dv\)
  • if \(\int vdu\) part is the same with question regardless of the coefficient, add/ subtract both side to eliminate the one on right side

$$\int x^2e^(\frac{x}{2})dx$$

Select u and dv: $$u = x^2, dv = e^(\frac{x}{2}) dx$$ (Select Algebra(A) as u first)
$$du = 2x, v = 2e^(\frac{x}{2})$$
$$\int x^2e^(\frac{x}{2})dx = uv - \int vdu$$
$$\int x^2e^(\frac{x}{2})dx = x^2*2e^(\frac{x}{2}) - \int 4xe^(\frac{x}{2})dx$$
Select u and dv: $$u = 2x, dv = 2e^(\frac{x}{2}) dx$$ (Select Algebra(A) as u first)
$$du = 2, v = 4e^(\frac{x}{2})$$
$$\int 4xe^(\frac{x}{2})dx = uv - \int vdu$$
$$\int 4xe^(\frac{x}{2})dx = 8e^(\frac{x}{2}) - \int 8e^(\frac{x}{2})dx$$
$$\int 4xe^(\frac{x}{2})dx = 8e^(\frac{x}{2}) - 16e^(\frac{x}{2}) + C$$
$$\int x^2e^(\frac{x}{2})dx = x^2*2e^(\frac{x}{2}) - 8e^(\frac{x}{2}) + 16e^(\frac{x}{2}) + C$$

$$\int x^5\sqrt[2]{x^3+1}$$

Select u and dv: $$u = x^3, dv = x^2\sqrt[2]{x^3+1} dx$$ (Exception)
$$du = 3x^2dx, v = \frac{2}{9}*(x^3+1)^(\frac{3}{2})$$
$$\int x^5\sqrt[2]{x^3+1} = \frac{2}{9}*x^3*(x^3+1)^(\frac{3}{2}) - \frac{2}{3}\int (x^2)*(x^3+1)^(\frac{3}{2})dx$$
$$\int x^5\sqrt[2]{x^3+1} = \frac{2}{9}*x^3*(x^3+1)^(\frac{3}{2}) - \frac{4}{45} *(x^3+1)^(\frac{5}{2})$$

Partial Fraction

  • It's a process to make the integral become easier to solve by separate complicated fraction into smaller simpler part
$$\frac{3x+2}{x^2+x}$$
$$= \frac{3x+2}{x(x+1)}$$
$$= \frac{A}{x} + \frac{B}{x+1}$$
multiply by \(x(x+1)\): $$3x+2 = A(x+1) + Bx$$
Rearrange (Comparing coefficient): $$3x+2 = (A+B)x + A$$
$$A+B = 3, A=2$$
$$A = 2, B = 1$$
Put A and B back: $$= \frac{2}{x} + \frac{1}{x+1}$$
  • if the denominator has \(A^n\) while \(A\) is polynomial and \(n\) is whole number, we must have \(A^i\) as a new fraction; \(i\) is 1 to n

    • $$\frac{Z}{(x+1)(x+3)^3} = \frac{A}{(x+3)} +\frac{B}{(x+3)^2} + \frac{C}{(x+3)^3} + \frac{D}{x+1}$$

  • if degree of Numerator is more than degree of denominator, long division is required
  • To find the numerator of the partial fraction, there's two options
    • Substitute x
    • Comparing coefficient
  • Followed by Basic rule

$$\frac{4x^2-22x+7}{(2x+3)(x-2)^2}$$

$$\frac{4x^2-22x+7}{(2x+3)(x-2)^2} = \frac{A}{2x+3} + \frac{B}{x-2} + \frac{C}{(x-2)^2}$$
multiply both sides with the LCD\((2x+3)(x-2)^2\):$$4x^2-22x+7 = A(x-2)^2 + B(x-2)(2x+3) + C(2x+3)$$
substitute x = 2: $$-21 = 7C$$
$$ C = -3 $$
substitute x = \(-3/2\): $$49 = A(-\frac{7}{2})^2$$
$$ A = 4 $$
substitute x = \(0, A, \text{and} C\): $$7 = 7 - 6B$$
$$ B = 0 $$
$$\frac{4x^2-22x+7}{(2x+3)(x-2)^2} = \frac{4}{2x+3} + \frac{-3}{(x-2)^2}$$

Indeterminate Forms and L'hopital's Rule

  • Indeterminate Forms is the expression obtained by applying the limit, but doesn't provide sufficient information to determine the real value
  • Indeterminate form comes in two forms: \(\frac{0}{0}\) and \(\frac{\infty}{\infty}\)
  • to tackle these Indeterminate forms, we use L'hopital's rule

    • $$\lim_{x\to c} \frac{f(x)}{g(x)} = \lim_{x\to c} \frac{f'(x)}{g'(x)}$$

  • but it can't be used if \(g'(x) = 0\) for some point in the interval
  • Repeat the process until the limit is not Indeterminate anymore or can directly find the answer

$$\lim_{t \to \infty} \frac{5t^4-4t^2-1}{10-t-9t^3}$$

Check if it's Indeterminate: $$\lim_{t \to \infty} \frac{5t^4-4t^2-1}{10-t-9t^3} = \frac{+\infty}{-\infty}$$
Apply L'hopital's: $$\lim_{t \to \infty} \frac{5t^4-4t^2-1}{10-t-9t^3} = \lim_{t \to \infty} \frac{20t^3-8t}{-1-27t^2}$$
Check if it's Indeterminate (again): $$\lim_{t \to \infty} \frac{20t^3-8t}{-1-27t^2} = \frac{+\infty}{-\infty}$$
Apply L'hopital's: $$\lim_{t \to \infty} \frac{20t^3-8t}{-1-27t^2} = \lim_{t \to \infty} \frac{60t^2-8}{-54t}$$
Check if it's Indeterminate (again): $$\lim_{t \to \infty} \frac{60t^2-8}{-54t} = \frac{+\infty}{-\infty}$$
Apply L'hopital's: $$\lim_{t \to \infty} \frac{60t^2-8}{-54t} = \lim_{t \to \infty} \frac{120t}{-54}$$
Apply t:$$ \lim_{t \to \infty} \frac{120t}{-54} = \infty$$
